# Manas Technologies — Gladiance-style Starter

This branch contains a starter Next.js + Tailwind CSS scaffold inspired by gladiance.com (placeholders & free assets only).

Quick start:
1. npm install
2. npm run dev
3. Open http://localhost:3000

What I included:
- Next.js + TypeScript starter
- Tailwind CSS setup
- Basic responsive layout (Hero, Services placeholders)
- Framer Motion for simple animations
